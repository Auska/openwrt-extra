OpenWrt-Extra
=============

Some extra packages for OpenWrt

Copy openwrt-extra to Packages

```bash
./scripts/feeds update -a
./scripts/feeds install -a
```

the list of packages:
* dnscrypt-proxy
* dns2socks
* chinadns
* shadowsocks-libev
* ChinaDNS
* ShadowVPN
* REDSOCKS2
* gargoyle
* n2n_v2
* ngrokc
* vlmcsd
